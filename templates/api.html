API

Supported Response Types

Parts Horse supports either JSON or HTML responses.

For /search, ensure a JSON response by having no Accept header or setting it to application/json.

Endpoints

Endpoint Purpose
/search?q=:query Search for :query and return all matches.
/parts.json A list of all documented parts.
/parts/:part.json Information for :part.
/ds/:part Redirects to the datasheet for :part. This URL is included in /parts/:part responses.
